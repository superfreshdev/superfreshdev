My name is Marko (SuperfreshDEV) and I'm a Fullstack Software Developer from Germany.
I'm Focusing on developing modern Web-Plattforms with Angular and including Concepts of Artifically Intelligence on it.
My Development Motivation are for example building visual attractive web platforms such as online shops , data analytics dashboards , various types of management tools and some mini web games for example like Tetris, PacMan, ... .
I like the experience when the customers says, this is a tool which you can use without any understanding.
